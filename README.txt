Liquidación de Pines

Este es un sistema web de liquidación de pines, que permite a los usuarios ingresar información del vehículo y del cliente, calcular el valor de la liquidación y seleccionar un método de pago.
Contiene una validación de formatos de placas para carro y moto y, además calcula el costo total del servicio.

Tecnologías Utilizadas:
- React
- JavaScript
- CSS

Para su instalación y uso local:
1. Ubicate en la carpeta donde está ubicado el proyecto usando la terminal junto con este comando: cd C:\Usuarios\prueba-tecnica
2. Instala las dependencias desde la terminal de esta forma: npm install
3. Inicia la aplicación usando npm run dev
